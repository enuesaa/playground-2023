# duckdb-pandas

SQLiteみたいにファイルベースのDBらしい。  
おそらくデータ分析用途。pandas から import したりできる

- インタフェースはSQL
- [duckdb/duckdb-wasm](https://github.com/duckdb/duckdb-wasm) で wasm 化できる模様
- よくわからないが python client は duckdb 本体のリポジトリにあるのかな。
