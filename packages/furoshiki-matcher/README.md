# furoshiki-matcher

## usecase
たくさんあるもののなかから、どれかひとつを選ぶもの
### 1. routing
### 2. cli command, args mapping

## example: inspired by rtk query
~~~ts
import { Matcher } from 'furoshiki-matcher'

const matcher = new Matcher({
  definitions: {
    default: () => {},
    aaa: () => {},
    bbb: () => {},
  },
  matchHandler: () => {},
})

const result = matcher.match('aaa');
~~~
